# ClimateCast: Global Weather & Country Info 🌍⛅

A sleek, responsive web application that provides real-time country demographic data and current weather conditions. Built with a premium, modern dark mode UI.

## Features
- **Search by Country**: Instantly pull detailed geographical and demographic data for any country (Capital, Region, Population, Flag).
- **Real-Time Weather**: Fetches current weather down to temperature and wind speed based on the country's exact coordinates.
- **Premium UI**: Dark-themed, glassmorphism design with responsive grid layouts and smooth CSS micro-animations.
- **No API Keys Needed**: Built entirely utilizing open and free APIs.

## Technologies Used
- HTML5 & CSS3
- Vanilla JavaScript (ES6+)
- [REST Countries API](https://restcountries.com/) (for country data & geographic coordinates).
- [Open-Meteo API](https://open-meteo.com/) (for real-time weather forecasts).

## Installation & Usage
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/your-username/weather-application.git
   ```
2. Open `index.html` directly in your favorite web browser. That's it! No local server or build pipeline is required.
